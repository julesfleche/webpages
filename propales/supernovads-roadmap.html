Sprint 1
Fondation · Notion + Claude + Supabase
On pose les bases sans lesquelles rien d'autre ne tient. Back-office unifié, modèle de données, Claude connecté. C'est le sprint le plus long parce que c'est le plus structurant.
2 semaines ~3–4 jours homme

Livrables

  • ✓Back-office Notion unifié, templaté pour chaque client
  • ✓Portail client Notion (lecture seule, propre)
  • ✓Supabase setup · table performances (Meta + Google, journalier)
  • ✓n8n · push quotidien APIs → Supabase
  • ✓Claude connecté Notion + n8n · Skills de base
  • ✓Connexion reporting ads existant dans Claude (slides Meta + Google)
  • ✓Section calls + roadmap client dans chaque portail

Automatisation · Calls + Alertes
On automatise ce que vous faites à la main. Les calls se transcrivent tout seuls, les to-dos arrivent dans Notion, et les comptes qui bougent vous le font savoir.

  • ✓Fathom → Notion · transcripts + to-dos automatiques post-call

  • ✓Dashboard Notion · salle de contrôle cross-clients
