La video complete.

Marche a suivre

10 etapes. 30 minutes.

Chaque etape est independante. Si tu as deja un compte GitHub, passe directement a la suivante.

⚠️
Chaque utilisateur doit suivre ces etapes

Les skills sont conditionnes a un utilisateur Claude. Si plusieurs personnes de ton equipe utilisent les skills, chacune doit faire ce setup sur son propre compte Claude Desktop.

1
5 min

Telecharger Claude Desktop

Claude Desktop, c'est l'application que tu vas utiliser tous les jours. C'est different de claude.ai dans le navigateur : l'app desktop peut se connecter a tes outils, lire des fichiers, et installer des skills.

Disponible sur Mac et Windows
Gratuit a installer, mais il faut un compte Pro pour les skills (etape suivante)
Telecharger Claude Desktop →
2
3 min

Creer un compte Claude Pro

Les skills fonctionnent uniquement avec un compte Claude Pro (20$/mois). C'est le plan qui debloque la connexion aux outils externes et l'installation de skills.

Si ton entreprise a deja un plan Claude Team ou Enterprise, utilise ton compte existant
Souscrire a Claude Pro →
3
3 min

Creer un compte GitHub

GitHub, c'est la ou on stocke tes skills. Tu n'as pas besoin de savoir coder : tu ne feras jamais rien directement sur GitHub, c'est Claude qui le lit pour toi en arriere-plan.

Le plan gratuit suffit
Choisis un username simple — prenom-nom ou le nom de ta boite
Creer un compte GitHub →
4
1 min

Accepter l'invitation au repo

On t'a envoye un email d'invitation pour acceder a ton repo prive client-[ton-nom]. C'est le dossier qui contient tous tes skills. Ouvre l'email et clique sur "Accept invitation".

L'email vient de noreply@github.com — check tes spams si tu ne le vois pas
Pas recu ? Dis-le sur Slack et on te le renvoie immediatement
5
5 min

Visite guidee de Claude Desktop

Prends 5 minutes pour decouvrir l'interface avant de connecter tes outils.

La barre de conversation en bas : c'est la que tu tapes tes commandes
Settings > Skills : c'est la que tes skills installes apparaissent
Settings > Integrations : c'est ici qu'on va connecter tes outils aux prochaines etapes
6
3 min

Installer le skill fetch-skills

C'est le seul skill que tu installes manuellement. Tous les autres s'installent automatiquement grace a lui. On te l'envoie sur Slack.

Tu recois un fichier — tu cliques "Save as a skill" dans Claude Desktop
Le skill apparait dans Settings > Skills
Tu ne fais ca qu'une seule fois
7
3 min

Connecter le MCP GitHub

Le MCP (Model Context Protocol), c'est le pont entre Claude Desktop et tes outils. En connectant GitHub, Claude pourra lire tes skills depuis ton repo prive.

Marche a suivre
1Ouvre Claude Desktop → Settings → Integrations
2Clique Add custom connector
3Colle cette URL :
https://api.githubcopilot.com/mcp
4Connecte ton compte GitHub quand la fenetre d'autorisation s'affiche
Tu devrais voir GitHub dans Settings > Integrations avec un check vert
8
5 min

Connecter le MCP n8n

n8n, c'est le moteur d'automatisation qui tourne en arriere-plan. Certains skills declenchent des workflows n8n — onboarding, contrats, reportings...

Marche a suivre
1Ouvre n8n dans ton navigateur (URL fournie par OOO sur Slack)
2Va dans Settings (en bas a gauche)
3Active MCP (toggle ON)
4Copie la configuration JSON affichee
5Ouvre Claude Desktop → Settings → Developpeur
6Clique Modifier la configuration
7Colle la config, sauvegarde et relance Claude Desktop
Pas de stress, on t'accompagne sur cette etape si besoin
9
5 min

Connexion des outils metier

Selon ton scope, on connecte tes outils metier (Notion, Slack, Klaviyo, Google Drive...) directement dans Claude Desktop pour que tes skills puissent lire et ecrire dedans.

Comment connecter un outil
1Ouvre Claude Desktop → Settings → Integrations
2Trouve le connecteur de l'outil (Notion, Slack, etc.)
3Clique Connect et autorise l'acces
4Claude a maintenant acces a cet outil dans tes conversations
On te dira exactement quels outils connecter selon ta roadmap
10
2 min

Tester : "recupere mes skills"

Le moment de verite. Ouvre une nouvelle conversation dans Claude Desktop et tape :

recupere mes skills

Claude va lire ton repo GitHub, trouver tous les skills, et te les proposer un par un en "Save as a skill". Clique sur chacun pour les installer.

Si Claude ne trouve pas le repo, verifie que le MCP GitHub est connecte (etape 7)
Si un skill ne s'installe pas, fais une capture et envoie-la sur Slack

Tu es operationnel.

A partir de maintenant, tu utilises tes skills en tapant des commandes naturelles. Et a chaque sprint, on push de nouveaux skills dans ton repo. Tu retapes "recupere mes skills" et c'est a jour.

Questions

Des doutes ?

Les questions qu'on recoit le plus souvent.

Ca coute combien Claude Pro ?

+

20$/mois (environ 18,50 euros). C'est le seul cout supplementaire a ta charge. Tout le reste — n8n, GitHub, les skills — est inclus dans ton retainer OOO.

Je ne sais pas coder, c'est un probleme ?

+

Zero probleme. Tu ne toucheras jamais a du code. Les skills, c'est du langage naturel : tu tapes "onboarde ce client" et Claude fait le reste. GitHub et n8n sont configures par notre equipe, tu n'y accedes jamais directement.

Comment je sais si mes skills sont a jour ?

+

Tape "recupere mes skills" a n'importe quel moment. Claude detecte les nouveaux skills et te propose de les installer. On te previent aussi sur Slack a chaque nouveau sprint livre.

Pret a demarrer

Un probleme ? On est là.

Envoie un message sur Slack, on repond dans la journee.
