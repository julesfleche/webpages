Onboarding Client

Lance le setup complet d’un nouveau client en une commande : canaux Slack, dossier Drive, portail Notion depuis le template binôme, mail de bienvenue et CGP envoyées pour signature.

NotionSlackGoogle DriveGmailDocuSeal
Vidéo de démo
Ce que ça fait
1
Collecte les 6 infos client et résout le binôme dans Notion
2
Crée les canaux Slack externe (client) et interne (équipe)
3
Crée le dossier Google Drive et le partage avec le client
4
Crée le portail Notion client depuis le template binôme
5
Envoie le message de bienvenue Slack + mail Gmail
6
Déclenche automatiquement l’envoi des CGP via DocuSeal
Skill Claude
purplemail-onboarding-client
GitHub›
Triggers
"onboarder un client""nouveau client Purplemail""créer le portail [nom]"
Infos requises
nom_client · prenom · email · binome · siret · adresse
Tous obligatoires — Claude bloque si un champ manque
Étapes
  1. Collecter les 6 infos obligatoires
  2. Résoudre le binôme via Notion MCP → récupérer l’ID
  3. Afficher le récap complet → attendre validation "oui"
  4. Créer le portail Notion (template binôme) AVANT le workflow
  5. Chercher le workflow via n8n-mcp:search_workflows
  6. Exécuter le workflow avec les 7 champs en payload
  7. Afficher le résultat (Notion · Drive · Slack · CGP · Mail)

Installation

Comment tout installer chez Purplemail ?

Tous les skills livrés dans ce kit s’installent en une seule commande Claude.

Utilisez le skill fetch-skills.

fetch-skills est le skill bootstrap. Installé une seule fois manuellement, il récupère tous les autres depuis le repo GitHub client-purplemail en une commande.

Voir le kit de démarrage →
1
Assure-toi d’avoir le skill fetch-skills installé. Sinon, voir le Kit de démarrage fourni.
2
Ouvre une nouvelle conversation dans Claude Desktop et tape :
récupère mes skills
3
Claude trouve les 6 skills et te propose "Save as a skill" pour chacun.
✓
Tous les skills sont installés. Même commande au prochain sprint pour tout mettre à jour.
