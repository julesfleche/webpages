01 · Foundations

Principes de design.

Quatre règles qui guident toutes les décisions visuelles OOO.

Dark first, toujours

Fond principal #080808. Les sections blanches servent de respiration, jamais de base.

Une seule couleur d'accent

Le bleu violet #5b6af0. Gradient avec violet/rose pour les hover et CTA principaux.

Typo à deux niveaux

Poppins 600 pour les titres, DM Sans pour le body. Letter-spacing négatif systématique.

Zéro AI vibes

Pas de tirets cadratins, pas d'emojis dans les titres. L'accent italique dégradé remplace le bold jaune.

14 · Copy & Tone

Voice OOO.

Le copy est un élément de design. Règles universelles : site, landing, email, slides.

✓ Do
  • • Tutoiement systématique (tu / ton / ta)
  • • Phrases courtes, percutantes
  • • Italique gradient pour les mots-clés de titre
  • • Virgules de rythme : "On fait. On livre. On mesure."
  • • Verbes d'action au présent
  • • Noms propres (n8n, Claude, Notion) cités tels quels
✗ Don't
  • • Tirets cadratins trop "AI-generated"
  • • Emojis dans les titres
  • • "Nous" formel / vouvoiement
  • • Buzzwords creux ("synergie", "disruptif")
  • • Promesses quantifiées non-sourcées
  • • Title case anglais
